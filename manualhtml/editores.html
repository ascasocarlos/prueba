Editores

Para hacer una pagina web simplemente necesitas un editor de texto simple como el bloc de notas pero hay editores mucho mas completos que ayudan a la hora de escribir las etiquetas con autocompletados o con la visibilidad cambiando los colores de dichas etiquetas para una mejor localizacion. Hay tanto editores locales como editores online, aqui algunos de ellos:

  • Atom: https://atom.io/
  • Brackets: https://brackets.io/
  • Sublime Text: https://www.sublimetext.com/
  • Blue Griffon: http://www.bluegriffon.org/
  • Editores online: W3Schools, TinyMCE, HTML online
