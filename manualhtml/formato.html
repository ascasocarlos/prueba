Formato

El formato del HTML esta compuesto por etiquetas formadas por los simbolos de mayor y menor que con el elemento entre dichos simbolos: <elemento>

Una vez abierta una etiqueta hay que cerrarla volviendo a escribir la etiqueta con una barra despues del simbolo menor que: </elemento>

Las etiquetas pueden tener atributos que modifiquen sus caracteristas, para escribirlos hay que usar el siguiente formato: <elemento atributo="valor"></elemento>

Todas las paginas HTML tienen que tener la misma estructura compuesta por etiquetas que indican sus partes. La estructura es la siguiente:

	<!doctype html> 				Tipo de documento
	<html lang="es">				Inicio y lenguaje
		<head> 					Cabecera
			<title></title> 		Titulo
			<meta> 				Metadatos (codificacion, autor, etc...)
		</head> 
		<body> 					Cuerpo
		</body> 
	</html> 					Fin

En una pagina como puedes ver hay diferentes secciones como la cabecera y el cuerpo, tambien existen secciones que se pueden añadir a la pagina pero que no son obligatorias. Podriamos añadir las siguientes(todas ellas llevan etiqueta de cierre):

Navegacion(menu): <nav>

Pie de pagina: <footer>

Articulos: <article>

Secciones: <section>

Esta ultima se puede utilizar introduciendola dentro de <article> y a la vez introducir <article> dentro de <section>
