Servidores Sencillos

Las paginas web tienen que estar en un servidor web para poder acceder a ellas, aunque puedes abrirlas con un navegador es recomendable utilizar un servidor local para simular que la pagina que estas haciendo esta en un servidor real.

Tambien hay servidores remotos a los que puedes acceder para trabajar en ellos, aun asi es mucho mas facil utilizar uno de los siguientes servidores locales que puedes ejecutar en local para crear tu pagina web:

  • Python
    • Python 2: python -m SimpleHTTPServer [port]
    • Python 3: python -m http.server [port]
  • PHP
    • php -S localhost:[port]
  • Node.js
    • Browsersync: https://www.browsersync.io/
