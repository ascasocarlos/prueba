Etiquetas Basicas

Etiquetas basicas para hacer una pagina web:

Titulos: Se usan las etiquetas <h1> hasta <h6> para introducir titulos de mayor a menor tamaño.

Parrafo: La etiqueta <p> sirve para introducir un parrafo sin formato ya que elimina los espacios y las tabulaciones, en cambio con la etiqueta <pre> introduces un parrafo con espacios y tabulaciones.

Listas: Podemos usar dos tipos de listas, las ordenadas y las desordenadas. Para ello tenemos que abrir la lista que queramos usar <ul> para desordenadas y <ol> para ordenadas y dentro de cada una de ellas usaremos el elemento <li> tantas veces como elementos tenga la lista. Tambien existen las listas de definicion <dl> con el termino de definicion <dt> y los datos de la definicion <dd>.

Linea horizontal: <hr> introduce una linea horizontal para crear una separacion entre dos elementos.

Elementos semanticos: Abrimos y cerramos las siguiente etiquetas introduciendo dentro las palabras a las que queremos cambiarle el formato. <b> (negrita), <i> (cursiva), <u> (subrayado).

Salto de linea: La etiqueta <br> (sin cierre) sirve para introcir un salto de linea en un punto del texto o para introducir espacios horizontales en blanco.

Hiperenlace: Se usa la etiqueta <a> (de ancla) para hacer un enlace a un punto de su misma pagina web con el atributo id poniendo el mismo nombre en el ancla y en el elemento al que haces referencia. (Para hacer un enlace a otra pagina hay que usar el atributo href).

Imagenes: <img> sirve para introducir una imagen, el atributo src indica donde se encuentra esa imagen. Ademas podemos añadir atributos como width y height para modificar su anchura y altura.

Tablas: La etiqueta <table> define una tabla(El atributo border dentro de <table> define el borde de la tabla). El encabezado de la tabla se define con <th> justo despues de <table>. Hay que añadir tantos <tr> como filas que quieras que contenga la tabla y tantos <td> como columnas dentro de cada <tr> que quieras que tenga la tabla.

Contenedores: La etiqueta <div> es un contenedor de informacion en bloque el cual comienza en una nueva linea, ocupa todo el ancho disponible y puede contener otros elementos de bloque o en linea. La etiqueta <span> es otro tipo de contenedor que a diferencia del <div> comienza en la misma linea, solo ocupa el espacio necesario pero solo puede contener otros elementos en linea.


Todas estas etiquetas pueden contener atributos para especificar su funcionamiento y a la vez pueden ser modificadas usando CSS para cambiar su estilo.
